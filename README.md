# FUTURE_FS_01
🌐 Personal Portfolio Website
📌 Overview

A full-stack personal portfolio website built during my internship to showcase my projects, skills, and growth as a developer. It includes a responsive design and a contact form with backend data handling.

🚀 Features
Responsive UI (mobile, tablet, desktop)
Project showcase & about section
Contact form with database storage
🛠️ Tech Stack
Frontend: HTML, CSS, Bootstrap 5, JavaScript
Backend: PHP
Database: MySQL
📚 Key Learnings
Full-stack development (frontend + backend integration)
Form handling and database connectivity
Responsive design principles
Debugging real-world application issues
⚡ Challenges
Integrating frontend with PHP backend
Handling form validation and data storage
Ensuring cross-device responsiveness
